# 阿宝宇宙工作台初始化指令

用途：创建阿宝宇宙工作台，整合 GitHub 仓库、Notion 知识库、每日工作日志和 Codex 工作汇报。

## 执行前检查

* 检查当前文件夹是否为 GitHub 仓库。
* 检查 git remote 是否连接到 GitHub。
* 检查当前分支、提交记录、未提交文件。
* 在修改文件前先输出检查结果。

## 初始化内容

* 创建 `/workspace` 下的 11 个资料目录。
* 每个目录创建 README.md。
* 创建 `/workspace/notion_setup_plan.md`。
* 创建 `/workspace/11_codex_daily_reports/daily_report_template.md`。
* 创建或更新根目录 `AGENTS.md`。
* 如果无法直接操作 Notion，创建 `/workspace/notion_sync_placeholder.md`。
